# 🔒 Security Policy

## Reporting a Vulnerability

This operation is monitored 24/7 by Agent T.

If you have discovered a security vulnerability in any **openS Plan**, please follow the protocol:

### 🟢 Declassified Plans (A, B, C, D, F, H, I, K, L, N, O, P, Q, R, S)
- Open an issue with the label `security`
- Do **NOT** include exploit details in public issues
- Use the subject line: `[SECURITY] Plan {X} — Brief description`

### 🔴 Classified Plans (E, G, J, M, T, U, V)
- These plans are not accessible to unauthorized personnel
- If you somehow obtained access, consider yourself compromised
- Stand by for further instructions

## Response Time

| Severity | Response Time |
|----------|--------------|
| Critical | Immediate |
| High | < 24 hours |
| Medium | < 72 hours |
| Low | When Agent T. feels like it |

## Scope

All 22 openS Plans are in scope. The GitHub Pages Operations Center (tuth.site) is also in scope.
